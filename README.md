# DocxFromHtml

Sample service that demonstrates how to create a docx file from an HTML string. The example leverages the DocumentFormat.OpenXml and HtmlToOpenXml.dll nuget packages

To use the service, issue an HTTP POST to the /docx/create endpoint. The service will return a fully formatted Word Document in docx format.
```
POST /docx/create HTTP/1.1
Host: docxfromhtml.azurewebsites.net
Content-Type: application/x-www-form-urlencoded
```
